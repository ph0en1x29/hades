# Abstract

Large Language Models are increasingly deployed for automated alert triage in Security Operations Centers, processing thousands of SIEM alerts that overwhelm human analysts. This collapses a trust boundary that classic security pipelines maintain: attacker-controlled telemetry becomes model-interpretable, instruction-bearing context. We systematize this vulnerability, recently demonstrated by Neaves [2025]: adversaries embed prompt injection payloads in network traffic fields — HTTP headers, DNS queries, TLS certificates — that SIEM systems feed directly to triage models.

We present **Hades**, an open-source evaluation framework and benchmark for studying this threat. Hades constructs a benchmark of 12,147 rule-linked alerts across 27 MITRE ATT&CK techniques and specifies an evaluation protocol for 4 open-weight MoE models with different routing designs. The adversarial variant space comprises up to 1.4 million samples (12 injection vectors × 5 attack classes × 2 base encodings × 12,147 alerts), specified for evaluation under three attacker knowledge levels (black-box, gray-box, white-box). Cross-model evaluation awaits GPU allocation; this paper reports framework design, benchmark construction, SIEM survival validation, and preliminary defense feasibility.

We design two defense components with preliminary evidence: (1) a **correlator agent** prototype for multi-stage campaign aggregation; and (2) **output consistency checks** (behavioral invariants) evaluated on manually constructed perturbations. In targeted validation across 11 SIEM normalization rules, all 15 payload templates survived without semantic loss; keyword matching detected payloads in only 2 of 5 attack classes. On 50 hand-crafted triage outputs (not live model generations), invariants flagged 100% of direct misclassification (C1) and reasoning corruption (C3), 98% of attention hijacking (C4), with 0/50 false positives — though wide confidence intervals and single-family technique coverage (T1003.001) limit these to proof-of-concept sensitivity. Under a counterfactual what-if analysis with manually constructed triage decisions, Fox campaign scores would decrease by 53.7 points (95.7→42.0) — this illustrates potential impact, not an observed attack success rate. We release Hades as open-source tooling for evaluating LLM-based security automation.
