# Notion Setup for HADES

## Steps (15 minutes)

1. Go to https://notion.so → sign in or create free account
2. Create a new page: **HADES — Project Tracker**
3. Add an icon: 🔥 and a cover image (optional)
4. Create the pages below by pasting the content from this folder

## Page Structure

```
HADES — Project Tracker
├── 📊 Dashboard (paste from DASHBOARD.md)
├── 🧪 Experiments (create as database — paste from EXPERIMENTS_DB.md)
├── 📅 Milestones (create as timeline — paste from MILESTONES.md)
├── 📝 Meeting Notes (create as database — use MEETING_TEMPLATE.md for each)
├── ❓ Decisions & Questions (paste from DECISIONS.md)
└── 📚 Resources (paste from RESOURCES.md)
```

## Sharing

1. Click **Share** (top right)
2. Click **Publish to web** or **Invite** Prof Liu's email
3. Set to **Can view** (or **Can comment** if you want his notes inline)

## When to update

- After every advisor meeting → add meeting notes
- When an experiment status changes → update the database
- When a decision is made → move from Open to Decided
- Monthly → review milestones, update timeline
